# {{GUIDE_NAV_LOCK}}

{{GUIDE_NAV_LOCK}} 기능을 사용하면 특정 위치에 특정 글자를 고정 표시하여 안정적인 비교 기준을 확립할 수 있습니다. 잠긴 글자는 참조 글자의 무작위 배열에 영향을 받지 않습니다.

## 기본 개념

### 9칸 위치

```
| 6 | 7 | 8 |  ← 상단
| 3 | 4 | 5 |  ← 중앙
| 0 | 1 | 2 |  ← 하단
```

- **위치 4 (중앙 셀)**: 현재 편집 중인 글자 표시, **잠금 불가**
- **기타 8개 위치**: 특정 글자 잠금 가능

### 잠금 글자 vs 참조 글자

| 특성 | 잠금 글자 | 참조 글자 |
|-----|--------|--------|
| 위치 | 지정 위치에 고정 | 빈 위치에 무작위 배치 |
| 무작위 배열 | 영향 받지 않음 | 클릭할 때마다 재배열 |
| 저장 방식 | 위치별 저장 | 글자 목록 저장 |

## ⌘ 모드 조작

**⌘** 키를 누르고 있으면 잠금 편집 모드로 들어가며, 모든 잠금/잠금 해제 작업은 이 모드에서 수행됩니다. 들어가면 잠긴 셀이 시스템 강조 색상으로 하이라이트됩니다.

### 클릭 조작

| 조작 | 대상 위치 | 효과 |
|-----|---------|------|
| ⌘ + 클릭 | 잠기지 않은 위치 | 해당 위치의 참조 글자 잠금 |
| ⌘ + 클릭 | 잠긴 위치 | 해당 위치 잠금 해제 |
| ⌘ + 클릭 | 중앙 셀 | 잠금 기능 활성화/비활성화 상태 전환 |
| ⌘ + ⌥ + 클릭 | 모든 위치 | {{GLYPH_PICKER_TITLE_LOCK|}} 선택기 열기 |
| ⌘ + ⌥ + 클릭 | 중앙 셀 | 모든 잠금 해제 |
| ⌘ + ⇧ + 클릭 | 중앙 셀 | 현재 잠금을 {{PRESETS_WINDOW_TITLE|}}에 저장 |

### 드래그 조작

| 조작 | 원본 | 효과 |
|-----|------|------|
| ⌘ + 드래그 | 참조 글자 위치 | 대상 위치에 잠금 |
| ⌘ + 드래그 | 잠긴 위치 | 잠금 글자 이동 |
| ⌘ + ⌥ + 드래그 | 모든 위치 | 잠금 복사 (원본 유지) |

## 중앙 셀 설명

중앙 셀 (위치 4)은 항상 현재 편집 중인 글자를 표시하며, **잠글 수 없습니다**.

> [!TIP]
> **⌘ + 드래그**를 사용하여 중앙 셀의 글자를 다른 위치로 복사하고 해당 글자의 잠금 글자를 만들 수 있습니다.

## 잠금 글자 패널

잠금 글자 패널은 잠금 글자의 표시와 관리를 위한 독립적인 플로팅 창입니다.

### 열기 방법

- **툴바**: 잠금 버튼 ({{ICON_LOCK}}) 클릭
- **컨텍스트 메뉴**: 9칸 내부에서 오른쪽 클릭하고 "{{CONTEXT_MENU_SHOW_LOCKED_PANEL|}}" 선택

### 패널 조작

패널은 3×3 9칸 형식으로 모든 잠금 글자의 설정을 표시합니다:

| 조작 | 효과 |
|-----|------|
| 클릭 | 해당 셀 선택 |
| 더블 클릭 | 글리프 선택기를 열어 해당 위치 변경 |
| ⌘ + 클릭 | 다중 선택 (선택 상태 전환) |
| ⇧ + 클릭 | 사각형 선택 |
| ⌘ + A | 잠긴 모든 위치 전체 선택 |
| Delete | 선택한 모든 잠금 글자 삭제 |
| Enter | 글리프 선택기를 열어 선택한 위치 일괄 변경 |

### 중앙 셀 전환

패널의 중앙 셀은 잠금 아이콘을 표시하며, 클릭하면 잠금 기능의 활성화/비활성화 상태를 전환합니다:

- **활성화 시**: 잠금 아이콘 ({{ICON_LOCK}}) 표시
- **비활성화 시**: 잠금 해제 아이콘 ({{ICON_LOCK_OPEN}}) 표시

## 표시 제어

### 잠금 글자 패널 전환

툴바의 잠금 버튼 ({{ICON_LOCK}})으로 잠금 글자 패널의 표시를 제어합니다:

- **열려 있을 때**: 패널 표시, 툴팁은 "{{LOCKED_PANEL_TOGGLE_HIDE|}}"
- **닫혀 있을 때**: 패널 숨김, 툴팁은 "{{LOCKED_PANEL_TOGGLE_SHOW|}}"

### 잠금 데이터 영속성

잠금 글자 데이터는 서체 파일 (`GSFont.userData`)에 저장되며 파일과 함께 저장됩니다:

- 서체 파일을 닫고 다시 열면 잠금 설정이 복원됨
- 다른 서체 파일은 독립적인 잠금 설정을 가짐

## 사용 팁

### 비교 기준 확립

표준 글자 (예: "永")를 잠그고 다른 글자와의 조합 효과를 관찰합니다.

### 디자인 일관성 확인

완성된 여러 글자를 잠그고 새 디자인의 글자가 그들과 스타일이 일관되는지 확인합니다.

### 설정의 빠른 저장

잠금 글자 설정이 완료되면 다음 방법으로 {{PRESETS_WINDOW_TITLE|}}에 저장할 수 있습니다:

- 9칸의 중앙 셀을 **⌘ + ⇧ + 클릭**
- 잠금 글자 패널의 **저장 버튼** 클릭
