## {{LIGHT_TABLE}} 통합

{{NINE_BOX_VIEW}}는 서드파티 플러그인 **[{{LIGHT_TABLE}}](https://formkunft.com/light-table/)**와의 통합을 지원하여 9칸 내에서 다양한 버전의 글자 디자인을 비교할 수 있습니다.

### {{LIGHT_TABLE}}란?

{{LIGHT_TABLE}}는 독립적인 Glyphs 플러그인으로 서체 버전 비교 기능을 제공합니다. {{LIGHT_TABLE}}가 설치되고 해당 도구가 활성화되면 9칸 내에서 비교 버전의 글자를 볼 수 있습니다.

### 활성화 조건

9칸에서 비교 버전을 표시하려면 다음 조건을 충족해야 합니다:

1. **{{LIGHT_TABLE}} 설치**: {{GLYPHS_PLUGIN_MANAGER}}에서 {{LIGHT_TABLE}} 플러그인 설치
2. **{{LIGHT_TABLE}} 도구 활성화**: Glyphs 툴바에서 {{LIGHT_TABLE}} 비교 도구 선택
3. **비교 버전 선택**: {{LIGHT_TABLE}} 패널에서 비교할 버전 선택
4. **Shift 키 누르기**: Shift 키를 눌러 비교 모드 트리거

### 사용 방법

1. {{LIGHT_TABLE}}가 설치되고 활성화되었는지 확인
2. {{LIGHT_TABLE}} 패널에서 비교 버전 선택
3. **Shift** 키 누르고 있기
4. 9칸 내의 글자가 비교 버전으로 전환됨

> [!NOTE]
> **Shift 키 동작**: {{LIGHT_TABLE}}가 설치되지 않았거나 도구가 활성화되지 않은 경우 Shift를 누르고 있으면 **Solo 모드**로 들어가 참조 글자와 잠금 글자가 일시적으로 숨겨지고 현재 편집 중인 글자만 표시됩니다. 주변 셀의 오버레이 효과와 현재 글자 자체의 차이를 빠르게 비교하는 데 편리합니다.

## 키보드 단축키

### 9칸 내 조작

| 단축키 | 조작 | 설명 |
|-------|------|------|
| 클릭 | 무작위 배열 | 참조 글자를 무작위로 재배열 |
| Cmd + 클릭 | 잠금/잠금 해제 | 해당 위치의 잠금 상태 전환 |
| Cmd + 중앙 셀 클릭 | 잠금 기능 전환 | 잠금 기능 활성화/비활성화 |
| Cmd + Option + 클릭 | 글리프 선택기 | 글리프 선택기를 열어 잠금 글자 설정 |
| Cmd + Option + 중앙 셀 클릭 | 모든 잠금 해제 | 모든 위치의 잠금을 한 번에 해제 |
| Cmd + Shift + 중앙 셀 클릭 | 잠금 프리셋 저장 | 현재 잠금 설정을 {{PRESETS_WINDOW_TITLE|}}에 저장 |

### 드래그 조작

| 단축키 | 드래그 효과 |
|-------|---------|
| 없음 | 최상층에 표시되는 글자 교환 (9칸 내) / 삽입 (편집 뷰) |
| Cmd | 잠금 글자 조작 (이동 또는 잠금) |
| Option | 9칸 내: 글자 복사 (대상이 잠금 글자인 경우 잠금 해제); 편집 뷰로: 새 탭에서 글자 열기 |
| Shift | 새 탭에서 9칸 열기 |
| Cmd + Option | 잠금 글자 복사 (원본 유지) |

### 툴바 조작

툴바 버튼은 클릭으로 트리거되며, 일부 버튼은 마우스 호버 시 툴팁을 표시합니다:

| 버튼 | 기능 |
|-----|------|
| {{ICON_SUN}} / {{ICON_MOON}} | 다크/라이트 모드 전환 |
| {{ICON_PRESETS}} | {{PRESETS_WINDOW_TITLE|}} 패널 표시/숨기기 |
| {{ICON_LOCK}} | {{LOCKED_PANEL_TITLE|}} 패널 표시/숨기기 |
| {{ICON_REFERENCE}} | {{REFERENCE_PANEL_TITLE|}} 패널 표시/숨기기 |

### 패널 스냅

**Option** 키를 누른 채 패널 버튼을 클릭하면 패널을 메인 창 측면에 스냅할 수 있습니다:

| 버튼 | Option + 클릭 효과 |
|-----|-------------------|
| {{ICON_PRESETS}} | 메인 창 **왼쪽**에 스냅 |
| {{ICON_REFERENCE}} | 메인 창 **오른쪽 상단**에 스냅 |
| {{ICON_LOCK}} | 메인 창 **오른쪽 하단**에 스냅 |

> [!TIP]
> 스냅 기능을 사용하면 흩어진 패널을 메인 창 주변에 빠르게 재배치하여 정돈된 작업 공간 레이아웃을 만들 수 있습니다.

## 그리드 선 표시

그리드 선 기능은 9칸에 가이드라인을 표시하여 글자의 정렬과 비율을 확인하는 데 도움이 됩니다.

### 활성화

1. 9칸 내부에서 오른쪽 클릭
2. "{{CONTEXT_MENU_SHOW_GRID_LINES|}}" 선택

### 그리드 내용

그리드 선은 다음 가이드라인을 표시합니다:

- **{{GRID_EM_BOX}}**: 글자의 경계 상자
- **{{GRID_CENTER_LINE}}**: 수평 및 수직 중심선

### 비활성화

1. 9칸 내부에서 오른쪽 클릭
2. "{{CONTEXT_MENU_HIDE_GRID_LINES|}}" 선택

> [!NOTE]
> 그리드 선 설정은 자동으로 저장되며 재시작 후에도 유지됩니다.

## 드래그 고급 조작

### Shift + 드래그로 새 탭 열기

**Shift** 키를 누른 채 9칸에서 Glyphs 창으로 드래그하면 9칸 전체 내용을 새 탭에서 열 수 있어 편집 뷰에서 배열 효과를 빠르게 미리볼 수 있습니다.

## 특수 레이어 지원

Glyphs에서 비마스터 레이어를 편집할 때 9칸의 주변 셀은 해당 레이어 버전을 자동으로 동기화하여 표시하며, 특수 레이어와 주변 글자의 조합 효과를 실시간으로 미리볼 수 있습니다.

> [!TIP]
> 주변 셀을 특수 레이어를 따르지 않고 마스터 레이어에 유지하려면 "잠금 글자" 또는 "참조 글자" 기능을 사용하세요. 잠금 글자와 참조 글자는 마스터 레이어에 유지됩니다.

## 데이터 영속성

{{NINE_BOX_VIEW}}의 데이터는 두 위치에 저장됩니다:

| 저장 위치 | 내용 | 설명 |
|---------|------|------|
| 서체 파일 | 참조 글자, 잠금 글자 | 파일과 함께 저장, 파일 간 공유되지 않음 |
| 애플리케이션 설정 | 창 설정, {{PRESETS_WINDOW_TITLE|}}, 테마 등 | 전역 설정, 파일 간 사용 가능 |

> [!TIP]
> 가져오기/내보내기 기능을 사용하여 다른 컴퓨터 간에 {{PRESETS_WINDOW_TITLE|}}를 공유할 수 있습니다.
