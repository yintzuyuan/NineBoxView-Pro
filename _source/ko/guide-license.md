## 체험 기간

{{NINE_BOX_VIEW}}를 처음 열면 환영 창이 표시되며, "{{LICENSE_WELCOME_START_TRIAL|}}"을 선택하여 14일 무료 체험을 시작합니다.

| 단계 | 설명 |
|-----|------|
| 체험 중 | 모든 기능 제한 없이 사용 가능, 남은 일수는 정보 메뉴에 표시 |
| 체험 종료 | 라이선스 구매, 라이선스 키 입력, 또는 무료 버전으로 전환 가능 |

## 라이선스 구매

1. 정보 메뉴의 "{{INFO_MENU_PURCHASE|}}" 클릭
2. 결제 완료 후 라이선스 파일 (`.license`)이 이메일로 발송됨

**라이선스 플랜**: 한 번 구매로 영구 사용, 무료 업데이트 포함, 소유한 여러 컴퓨터에서 사용 가능.

## 라이선스 활성화

라이선스 구매 후 `.license` 라이선스 파일을 받게 됩니다. 활성화 방법:

1. 타이틀 바의 "{{ICON_MENU}}" 버튼을 클릭하고 "{{INFO_MENU_ENTER_LICENSE|}}" 선택
2. 라이선스 파일을 창에 드래그 앤 드롭 (힌트: "{{LICENSE_DROP_HINT|}}")
3. "{{LICENSE_DROP_ACTIVE|}}"가 표시되면 마우스 놓기
4. 검증 성공 후 "{{LICENSE_ACTIVATION_SUCCESS_TITLE|}}"가 표시되고 자동으로 활성화 완료

### 일반적인 오류

| 오류 메시지 | 설명 |
|---------|------|
| {{LICENSE_ERROR_INVALID_KEY|}} | 라이선스 키 형식이 올바르지 않거나 일치하지 않음 |
| {{LICENSE_ERROR_INVALID_FILE|}} | 라이선스 파일이 손상되었거나 유효하지 않음 |
| {{LICENSE_ERROR_WRONG_PRODUCT|}} | 라이선스 키가 이 제품에 적용되지 않음 |

## 라이선스 상태

"{{ICON_MENU}}" → "{{ABOUT_MENU_ITEM|}}"를 클릭하여 라이선스 상태와 버전 정보를 확인합니다.

## 자주 묻는 질문

### 여러 컴퓨터에서 사용할 수 있나요?

네, 소유한 여러 컴퓨터에서 동일한 라이선스 키를 사용할 수 있습니다. 새 컴퓨터에서 라이선스 파일을 다시 로드해야 합니다. 라이선스 파일은 만료되지 않으며 무제한으로 사용할 수 있습니다.

### 라이선스 키를 분실하면?

> [!TIP]
> [support@erikyin.net](mailto:support@erikyin.net)으로 이메일을 보내고 구매 시 사용한 이메일 주소를 제공해 주세요. 라이선스 키 복구를 도와드리겠습니다.

### 라이선스 데이터는 어디에 저장되나요?

라이선스 데이터는 컴퓨터에 저장됩니다. 온라인 상태에서는 라이선스 유효성만 검증하며 다른 사용 정보는 수집하지 않습니다. Glyphs를 다시 설치한 후 라이선스 파일을 다시 로드해야 합니다.
