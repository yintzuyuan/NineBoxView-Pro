## {{PRESETS_WINDOW_TITLE|}} 열기

{{PRESETS_WINDOW_TITLE|}}를 여는 방법은 두 가지입니다:

1. **툴바 버튼**: 툴바의 **{{ICON_PRESETS}}** 버튼 클릭
2. **컨텍스트 메뉴**: 9칸 내부에서 오른쪽 클릭하고 "{{CONTEXT_MENU_SHOW_PRESETS|}}" 선택

## 탭 설명

{{PRESETS_WINDOW_TITLE|}}에는 두 개의 탭이 있습니다:

| 탭 | 저장 내용 |
|-----|---------|
| {{PALETTE_TAB_REFERENCE|}} | 참조 글자의 글자 조합 |
| {{PALETTE_TAB_LOCKED|}} | 잠금 글자의 위치 설정 |

각 프리셋에는 이름, 내용, 고정 상태가 포함됩니다.

## 검색 및 정렬

### 검색 기능

탭 컨트롤 아래에 검색 필드가 있어 프리셋을 빠르게 필터링할 수 있습니다:

1. 검색 상자에 키워드 입력
2. 프리셋 목록이 실시간으로 필터링됨
3. 검색은 대소문자를 구분하지 않음
4. 검색 상자를 비우면 전체 목록이 복원됨

> [!TIP]
> 검색 결과가 없으면 "{{SEARCH_NO_RESULTS|}}" 힌트가 표시됩니다.

### 정렬 기능

패널 하단의 정렬 버튼을 클릭하여 정렬 방법을 선택할 수 있습니다:

| 정렬 옵션 | 설명 |
|---------|------|
| {{SORT_NAME_AZ|}} | 이름 알파벳 순서 |
| {{SORT_NAME_ZA|}} | 이름 역알파벳 순서 |
| {{SORT_DATE_NEWEST|}} | 최신 추가를 맨 위에 |
| {{SORT_DATE_OLDEST|}} | 가장 오래된 추가를 맨 위에 |

> [!NOTE]
> 정렬 상태는 자동으로 저장되며 재시작 후에도 유지됩니다.

## 프리셋 만들기

참조 글자 또는 잠금 글자를 설정한 후 다음 방법으로 프리셋으로 저장할 수 있습니다:

| 입구 | {{PALETTE_TAB_REFERENCE|}} | {{PALETTE_TAB_LOCKED|}} |
|-----|--------|--------|
| 전용 패널 | "{{SAVE_REFERENCE_BUTTON|}}" 버튼 클릭 | "{{SAVE_LOCKED_BUTTON|}}" 버튼 클릭 |
| {{PRESETS_WINDOW_TITLE|}} | 해당 탭으로 전환, {{ICON_PLUS}} 클릭 | 해당 탭으로 전환, {{ICON_PLUS}} 클릭 |
| 단축키 | — | ⌘ + ⇧ + 중앙 셀 클릭 |

> [!TIP]
> 새로 만든 프리셋은 내용을 기반으로 자동으로 이름이 지정되며 나중에 이름을 변경할 수 있습니다.

## 프리셋 관리

| 조작 | 단축키 | 컨텍스트 메뉴 |
|-----|--------|---------|
| 로드 | 프리셋 더블 클릭 | — |
| 이름 변경 | 선택 후 Enter 누르기 | {{PALETTE_MENU_RENAME|}} |
| 덮어쓰기 | — | {{PALETTE_MENU_OVERWRITE|}} |
| 삭제 | 선택 후 Delete 누르기 | {{PALETTE_MENU_DELETE|}} |

> [!TIP]
> 다중 선택: ⌘ + 클릭 (선택 전환) 또는 ⇧ + 클릭 (범위 선택).

> [!CAUTION]
> 덮어쓰기 작업은 되돌릴 수 없습니다. 실행 전에 확인하세요.

## 고정 기능

고정된 프리셋은 목록 상단에 고정되어 표시됩니다.

| 조작 | 컨텍스트 메뉴 |
|-----|---------|
| 고정 | {{PALETTE_MENU_PIN|}} |
| 일괄 고정 | {{PALETTE_MENU_PIN_MULTIPLE|}} |
| 고정 해제 | {{PALETTE_MENU_UNPIN|}} |

고정된 프리셋은 항상 맨 위에 표시되며, 고정되지 않은 프리셋은 정렬 설정에 따라 배치됩니다.

## 가져오기 및 내보내기

패널 하단의 **{{ICON_MORE}}** 버튼을 클릭하여 가져오기/내보내기를 수행합니다:

| 조작 | 메뉴 항목 |
|-----|---------|
| 선택 내보내기 | {{PRESETS_MENU_EXPORT_SELECTED|}} |
| 전체 내보내기 | {{PRESETS_MENU_EXPORT_ALL|}} |
| 가져오기 | {{PRESETS_MENU_IMPORT|}} |

### 파일 형식

| 탭 | 확장자 |
|-----|--------|
| {{PALETTE_TAB_REFERENCE|}} | `{{FILE_EXTENSION_REFERENCE|}}` |
| {{PALETTE_TAB_LOCKED|}} | `{{FILE_EXTENSION_LOCKED|}}` |

가져오기 시 추가 ({{PRESETS_IMPORT_MODE_APPEND|}}) 또는 교체 ({{PRESETS_IMPORT_MODE_REPLACE|})를 선택할 수 있습니다.

> [!NOTE]
> 내보내기 시 고정 상태는 유지되지 않습니다. 가져오기 대화 상자는 해당 탭의 파일 유형을 자동으로 필터링합니다.

## 사용 팁

### 다양한 언어용 프리셋 만들기

여러 테스트 프리셋을 만듭니다. 예:

- "한글 상용": 가나다라마
- "중국어 번체": 永東國酬鷹
- "일본어": あいうえお

### 다양한 프로젝트용 설정 만들기

가져오기/내보내기 기능을 사용하여 다른 컴퓨터나 프로젝트 간에 프리셋을 공유합니다:

1. 한 컴퓨터에서 자주 사용하는 프리셋 만들기
2. 파일로 내보내기
3. 다른 컴퓨터에서 가져오기

### 잠금 글자와 결합하여 완전한 테스트 환경 만들기

1. {{PALETTE_TAB_REFERENCE|}} 프리셋 만들기: 자주 사용하는 참조 글자
2. {{PALETTE_TAB_LOCKED|}} 프리셋 만들기: 표준 글자 위치 설정
3. 두 가지를 빠르게 로드하여 완전한 테스트 환경 구축
