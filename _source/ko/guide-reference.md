# {{GUIDE_NAV_REFERENCE}}

{{GUIDE_NAV_REFERENCE}} 기능을 사용하면 글자 세트를 입력하여 9칸 주변에 표시하고, 현재 편집 중인 글자가 다양한 컨텍스트에서 어떤 시각적 효과를 가지는지 관찰할 수 있습니다.

## {{REFERENCE_PANEL_TITLE|}} 열기

{{REFERENCE_PANEL_TITLE|}}를 여는 방법은 두 가지입니다:

1. **툴바 버튼**: 툴바의 {{ICON_REFERENCE}} 버튼 클릭
2. **컨텍스트 메뉴**: 9칸 내부에서 오른쪽 클릭하고 "{{CONTEXT_MENU_SHOW_REFERENCE_INPUT|}}" 선택

패널이 열리면 텍스트 입력 영역이 표시되며, 여기에 참조 글자를 입력할 수 있습니다.

## 입력 형식

{{REFERENCE_PANEL_TITLE|}}는 다양한 입력 형식을 지원합니다:

### 직접 글자 입력

글자를 직접 입력하면 모든 글자가 한 글자씩 분석됩니다:

```
한가나다라마
```

glyph name을 지정하려면 `/`로 시작하세요 (아래 참조).

### Glyph Name 형식

슬래시 `/`로 시작하여 글자 이름을 지정합니다:

```
/adieresis /Agrave /uni6C38
```

### Unicode 형식

다양한 Unicode 입력 방법을 지원합니다:

| 형식 | 예시 | 설명 |
|-----|------|------|
| `/uniXXXX` | `/uni6C38` | 4자리 16진수 (공식 형식) |
| `/uXXXXX` | `/u06C38` | 5자리 16진수 |
| `/XXXX` | `/6C38` | 4-5자리 숫자만 (편의 형식) |

### 혼합 입력

다양한 형식을 혼합하여 사용할 수 있습니다:

```
한 /adieresis 가 /uni0041 나
```

## 조작 방법

### 참조 글자 저장

입력 완료 후 "{{SAVE_REFERENCE_BUTTON|}}" 버튼을 클릭하여 {{PRESETS_WINDOW_TITLE|}}에 저장합니다.

### 무작위 배열

9칸 내부를 클릭하면 (보조 키 없이) 참조 글자가 주변 위치에 무작위로 재배열됩니다.

> [!NOTE]
> 중앙 셀 (위치 4)은 항상 현재 편집 중인 글자를 표시하며 참조 글자로 대체되지 않습니다.

## 컨텍스트 메뉴

{{REFERENCE_PANEL_TITLE|}} 입력 영역에서 오른쪽 클릭하면 컨텍스트 메뉴가 열립니다:

| 항목 | 설명 |
|-----|------|
| {{CONTEXT_MENU_GLYPH_PICKER|}} | 글리프 선택기를 열어 여러 글자를 선택하여 삽입 |
| {{CONTEXT_MENU_INSERT_AT_CURSOR|}} | 참조 글자를 Glyphs 편집 뷰의 커서 위치에 삽입 |
| {{CONTEXT_MENU_OPEN_IN_NEW_TAB|}} | 참조 글자를 새 탭에서 열기 |

## 드래그 앤 드롭

### 기본 드래그

9칸 내의 글자를 다른 위치로 드래그:

| 조작 | 효과 |
|-----|------|
| 일반 드래그 | 위치 교환 |
| Option + 드래그 | 참조 글자 복사 (원본 유지) |

### Glyphs 편집 뷰로 드래그

9칸 내의 글자를 Glyphs의 편집 뷰로 드래그:

| 조작 | 효과 |
|-----|------|
| 일반 드래그 | 커서 위치에 글자 삽입 |
| Option + 드래그 | 해당 글자를 새 탭에서 열기 |

### {{REFERENCE_PANEL_TITLE|}}로 드래그

9칸 내의 글자를 입력 영역으로 드래그하여 참조 글자를 빠르게 추가:

- 9칸 내 어떤 위치의 글자도 허용
- 글자가 커서 위치에 삽입됨
- 앞뒤에 자동으로 공백 추가

> [!TIP]
> 기타 드래그 조작 (Cmd 모드, Shift 모드 등)은 [고급 기능](advanced.html#드래그-고급-조작)을 참조하세요.

## 사용 팁

### 빠른 테스트 그룹

1. 자주 사용하는 테스트 글자 입력 (예: "한가나다")
2. "{{SAVE_REFERENCE_BUTTON|}}"를 클릭하여 {{PRESETS_WINDOW_TITLE|}}에 저장
3. 나중에 {{PRESETS_WINDOW_TITLE|}}에서 빠르게 로드

### 간격 확인

동일한 글자의 배열 효과를 관찰하려면 단일 글자만 입력하면 됩니다:

```
가글
```

시스템이 9칸의 각 위치에 해당 글자를 자동으로 반복 표시하여 자간과 시각적 균형을 쉽게 관찰할 수 있습니다.
