## Trial Period

### First Launch

When opening {{NINE_BOX_VIEW}} for the first time, a welcome window appears:

**{{LICENSE_WELCOME_TITLE|}}**

You can choose:

| Option | Description |
|--------|-------------|
| {{LICENSE_WELCOME_START_TRIAL|}} | Start 14-day free trial |
| {{LICENSE_WELCOME_ENTER_KEY|}} | Enter license key to activate |

### During Trial

> [!NOTE]
> During the trial period, you have access to all features of {{NINE_BOX_VIEW}} without any restrictions.

Trial status is shown in the info menu, for example:

> {{LICENSE_TRIAL_REMAINING|}}

### Trial Expiration

When the 14-day trial ends, a prompt window appears:

**{{LICENSE_EXPIRED_TITLE|}}**

**{{LICENSE_EXPIRED_MESSAGE|}}**

You can choose:

| Option | Description |
|--------|-------------|
| {{LICENSE_EXPIRED_PURCHASE|}} | Go to purchase page |
| {{LICENSE_EXPIRED_ENTER_KEY|}} | Enter purchased license key |
| {{LICENSE_EXPIRED_FREE_VERSION|}} | Use free version (NineBoxView) |

## Purchasing a License

### How to Purchase

1. Click "{{INFO_MENU_PURCHASE|}}" in the info menu
2. Or visit the official website purchase page
3. After payment, the license file (`.license`) will be sent to your email

### License Type

{{NINE_BOX_VIEW}} offers a single license option:

- One-time purchase, use forever
- Includes free updates
- Can be used on multiple computers you own

## Activating Your License

After purchasing, you'll receive a `.license` file. To activate:

1. Click the "{{ICON_MENU}}" button in the title bar, select "{{INFO_MENU_ENTER_LICENSE|}}"
2. Drag and drop the license file into the window (prompt: "{{LICENSE_DROP_HINT|}}")
3. When you see "{{LICENSE_DROP_ACTIVE|}}", release the mouse
4. After successful verification, activation completes automatically

### License Input Window

**{{LICENSE_INPUT_TITLE|}}**

The center of the window is a drop zone â€” drag the `.license` file here to activate.

Bottom buttons:

- **{{LICENSE_INPUT_CONTACT|}}**: Contact support
- **{{LICENSE_INPUT_BUY|}}**: Go to purchase page
- **{{LICENSE_INPUT_CANCEL|}}**: Cancel

### Activation Success

After successful verification, "{{LICENSE_ACTIVATION_SUCCESS_TITLE|}}" appears in the center of the screen and automatically disappears after 2 seconds.

## License Status

### View License Status

License status is shown in the "{{ABOUT_WINDOW_TITLE|}}" window. To open:

1. Click the "{{ICON_MENU}}" button in the title bar
2. Select "{{ABOUT_MENU_ITEM|}}"

### About Window Content

The "{{ABOUT_WINDOW_TITLE|}}" window displays:

- **Plugin icon and name**
- **Version number**
- **License status**:
  - Trial: Shows remaining days
  - Licensed: Shows licensee name and email
- **Official website link**
- **Copyright notice**

### License Data Storage

License data is securely stored on your computer:

- Not uploaded to any server
- No internet connection required for use
- License file needs to be reloaded after reinstalling Glyphs

### Switch to Free Version

1. In the trial expired window, select "{{LICENSE_EXPIRED_FREE_VERSION|}}"
2. Or install "NineBoxView" (free version) from {{GLYPHS_PLUGIN_MANAGER}}
3. Remove "NineBoxView Pro" plugin

### Upgrade to Pro

To upgrade from free to Pro:

1. Purchase a license
2. Install "NineBoxView Pro" from {{GLYPHS_PLUGIN_MANAGER}}
3. Enter license key to activate
4. Remove free version plugin (optional)

## FAQ

### Can I use it on multiple computers?

Yes, you can use the same license key on multiple computers you own.

### Do I need to reactivate after changing computers?

Yes, you need to reload the license file on a new computer. The license file never expires and can be used unlimited times.

### Can the trial period be extended?

Sorry, the trial period cannot be extended. 14 days should be sufficient to fully experience all features.

### What if I lose my license key?

> [!TIP]
> Please email [support@erikyin.net](mailto:support@erikyin.net) with the email address used for purchase, and we'll help you recover your license key.
