# {{GUIDE_NAV_LOCK}}

{{GUIDE_NAV_LOCK}}功能讓您將特定位置固定顯示某個字符，建立穩定的比較基準。鎖定的字符不會被參考字的亂數排列影響。

## 基本概念

### 九宮格位置

```
| 6 | 7 | 8 |  ← 頂部
| 3 | 4 | 5 |  ← 中央
| 0 | 1 | 2 |  ← 底部
```

- **位置 4（中央格）**：顯示當前編輯的字符，**不可鎖定**
- **其他 8 個位置**：可以鎖定特定字符

### 鎖定字 vs 參考字

| 特性 | 鎖定字 | 參考字 |
|-----|--------|--------|
| 位置 | 固定在指定位置 | 隨機分配到空位 |
| 亂數排列 | 不受影響 | 每次點擊重新排列 |
| 儲存方式 | 依位置儲存 | 儲存字符清單 |

## ⌘ 模式操作

按住 **⌘** 鍵進入鎖定編輯模式，所有鎖定/解鎖操作都在此模式下進行。進入時，已鎖定的格子會以系統強調色高亮顯示。

### 點擊操作

| 操作 | 目標位置 | 效果 |
|-----|---------|------|
| ⌘ + 點擊 | 未鎖定位置 | 鎖定該位置的參考字 |
| ⌘ + 點擊 | 已鎖定位置 | 解除該位置的鎖定 |
| ⌘ + 點擊 | 中央格 | 切換鎖定功能的啟用/停用狀態 |
| ⌘ + ⌥ + 點擊 | 任意位置 | 開啟{{GLYPH_PICKER_TITLE_LOCK|}}選擇器 |
| ⌘ + ⌥ + 點擊 | 中央格 | 清除所有鎖定 |
| ⌘ + ⇧ + 點擊 | 中央格 | 儲存當前鎖定到{{PRESETS_WINDOW_TITLE|}} |

### 拖曳操作

| 操作 | 來源 | 效果 |
|-----|------|------|
| ⌘ + 拖曳 | 參考字位置 | 鎖定到目標位置 |
| ⌘ + 拖曳 | 已鎖定位置 | 移動鎖定字 |
| ⌘ + ⌥ + 拖曳 | 任意位置 | 複製鎖定（來源保留） |

## 中央格說明

中央格（位置 4）始終顯示當前編輯的字符，**無法被鎖定**。

> [!TIP]
> 使用 **⌘ + 拖曳**可將中央格字符複製到其他位置，建立該字符的鎖定字。

## 鎖定字面板

鎖定字面板是獨立的浮動視窗，專門用於顯示和管理鎖定字。

### 開啟方式

- **工具列**：點擊鎖定按鈕（{{ICON_LOCK}}）
- **右鍵選單**：在九宮格內容區按右鍵，選擇「{{CONTEXT_MENU_SHOW_LOCKED_PANEL|}}」

### 面板操作

面板以 3×3 九宮格形式顯示所有鎖定字的配置：

| 操作 | 效果 |
|-----|------|
| 點擊 | 選取該格子 |
| 雙擊 | 開啟字符選擇器修改該位置 |
| ⌘ + 點擊 | 多選（切換選取狀態） |
| ⇧ + 點擊 | 矩形選取 |
| 方向鍵 | 在格子間移動選取 |
| ⇧ + 方向鍵 | 擴展選取範圍（類似 Finder） |
| ⌘ + A | 全選所有已鎖定的位置 |
| Delete | 刪除所有選取的鎖定字 |
| Enter | 開啟字符選擇器批量修改選取的位置 |

### 中央格切換

面板的中央格顯示鎖頭圖示，點擊可切換鎖定功能的啟用/停用狀態：

- **啟用時**：顯示鎖定圖示（{{ICON_LOCK}}）
- **停用時**：顯示解鎖圖示（{{ICON_LOCK_OPEN}}）

## 顯示控制

### 切換鎖定字面板

工具列的鎖定按鈕（{{ICON_LOCK}}）控制鎖定字面板的顯示：

- **已開啟**：面板顯示，提示文字為「{{LOCKED_PANEL_TOGGLE_HIDE|}}」
- **已關閉**：面板隱藏，提示文字為「{{LOCKED_PANEL_TOGGLE_SHOW|}}」

### 鎖定資料持久化

鎖定字資料儲存在字型檔案中（`GSFont.userData`），會隨檔案保存：

- 關閉並重新開啟字型檔案後，鎖定設定會被恢復
- 不同字型檔案有獨立的鎖定設定

## 使用技巧

### 建立比較基準

鎖定一個標準字符（如「永」），觀察其他字符與它的搭配效果。

### 檢查設計一致性

鎖定多個已完成的字符，確保新設計的字符與它們風格一致。

### 快速儲存配置

設定好鎖定字配置後，可透過以下方式儲存到{{PRESETS_WINDOW_TITLE|}}：

- **⌘ + ⇧ + 點擊**九宮格中央格
- 點擊鎖定字面板的**儲存按鈕**
