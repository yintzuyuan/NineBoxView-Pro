# {{GUIDE_NAV_REFERENCE}}

{{GUIDE_NAV_REFERENCE}}功能让您输入一组字符，在九宫格周围显示，帮助观察当前编辑字符在不同上下文中的视觉效果。

## 打开{{REFERENCE_PANEL_TITLE|}}

有两种方式打开{{REFERENCE_PANEL_TITLE|}}：

1. **工具栏按钮**：点击工具栏的 {{ICON_REFERENCE}} 按钮
2. **右键菜单**：在九宫格内按右键，选择「{{CONTEXT_MENU_SHOW_REFERENCE_INPUT|}}」

面板打开后会显示一个文字输入区，您可以在此输入参考字符。

## 输入格式

{{REFERENCE_PANEL_TITLE|}}支持多种输入格式：

### 直接输入字符

直接输入字符，所有字符都会逐字拆解：

```
永东国酬鹰
```

若需指定 glyph name，请使用 `/` 开头（见下方说明）。

### Glyph Name 格式

使用斜线 `/` 开头指定字符名称：

```
/adieresis /Agrave /uni6C38
```

### Unicode 格式

支持多种 Unicode 输入方式：

| 格式 | 示例 | 说明 |
|-----|------|------|
| `/uniXXXX` | `/uni6C38` | 4 位十六进制（官方格式） |
| `/uXXXXX` | `/u06C38` | 5 位十六进制 |
| `/XXXX` | `/6C38` | 纯数字 4-5 位（便利格式） |

### 混合输入

可以混合使用不同格式：

```
永 /adieresis 东 /uni0041 国
```

## 操作方式

### 保存参考字

输入完成后，点击「{{SAVE_REFERENCE_BUTTON|}}」按钮保存到{{PRESETS_WINDOW_TITLE|}}。

### 替换粘贴

点击「{{PASTE_AND_REPLACE_BUTTON|}}」按钮会清空输入区并粘贴剪贴板内容。

### 随机排列

在九宫格内容区点击（不按任何辅助键），参考字会随机重新排列到周围位置。

> [!NOTE]
> 中央格（位置 4）始终显示当前编辑的字符，不会被参考字取代。

## 右键菜单

在{{REFERENCE_PANEL_TITLE|}}输入区按右键可打开快捷菜单：

| 项目 | 说明 |
|-----|------|
| {{CONTEXT_MENU_GLYPH_PICKER|}} | 打开字符选择器，可多选字符插入 |
| {{CONTEXT_MENU_INSERT_AT_CURSOR|}} | 将参考字插入到 Glyphs 编辑视图的光标位置 |
| {{CONTEXT_MENU_OPEN_IN_NEW_TAB|}} | 将参考字在新分页中打开 |

### 字符选择器

选择「{{CONTEXT_MENU_GLYPH_PICKER|}}」会打开 Glyphs 内置的字符选择器：

1. 在选择器中选取一个或多个字符
2. 确认后，字符会以 glyph name 格式插入到输入区
3. 插入时会自动在前后加入空格分隔

## 拖曳功能

### 基本拖曳

从九宫格内的字符拖曳到其他位置：

| 操作 | 效果 |
|-----|------|
| 普通拖曳 | 交换位置 |
| Option + 拖曳 | 复制参考字（来源保留） |

### 拖曳到 Glyphs 编辑视图

将九宫格内的字符拖曳到 Glyphs 的编辑视图：

| 操作 | 效果 |
|-----|------|
| 普通拖曳 | 字符插入到光标位置 |
| Option + 拖曳 | 在新分页中打开该字符 |

### 拖曳到{{REFERENCE_PANEL_TITLE|}}

将九宫格内的字符拖曳到输入区，可快速添加参考字：

- 接受九宫格内任何位置的字符
- 字符会插入到光标位置
- 自动在前后加入空格分隔

> [!TIP]
> 更多拖曳操作（如 Cmd 模式、Shift 模式）请参阅[进阶功能](advanced.html#拖曳进阶操作)。

## 使用技巧

### 快速测试字组

1. 输入常用的测试字组（如「永东国酬」）
2. 点击「{{SAVE_REFERENCE_BUTTON|}}」保存到{{PRESETS_WINDOW_TITLE|}}
3. 之后可从{{PRESETS_WINDOW_TITLE|}}快速加载

### 观察不同密度

尝试使用不同笔画密度的字符作为参考：

```
一二三 酬鹰郁
```

### 检查间距

想要观察相同字符的排列效果时，只需输入单一字符：

```
国今
```

系统会自动在九宫格的各个位置重复显示该字符，方便观察字距和视觉平衡。
