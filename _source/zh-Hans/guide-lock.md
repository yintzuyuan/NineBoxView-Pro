# {{GUIDE_NAV_LOCK}}

{{GUIDE_NAV_LOCK}}功能让您将特定位置固定显示某个字符，建立稳定的比较基准。锁定的字符不会被参考字的随机排列影响。

## 基本概念

### 九宫格位置

```
| 6 | 7 | 8 |  ← 顶部
| 3 | 4 | 5 |  ← 中央
| 0 | 1 | 2 |  ← 底部
```

- **位置 4（中央格）**：显示当前编辑的字符，**不可锁定**
- **其他 8 个位置**：可以锁定特定字符

### 锁定字 vs 参考字

| 特性 | 锁定字 | 参考字 |
|-----|--------|--------|
| 位置 | 固定在指定位置 | 随机分配到空位 |
| 随机排列 | 不受影响 | 每次点击重新排列 |
| 保存方式 | 依位置保存 | 保存字符列表 |

## ⌘ 模式操作

按住 **⌘** 键进入锁定编辑模式，所有锁定/解锁操作都在此模式下进行。进入时，已锁定的格子会以系统强调色高亮显示。

### 点击操作

| 操作 | 目标位置 | 效果 |
|-----|---------|------|
| ⌘ + 点击 | 未锁定位置 | 锁定该位置的参考字 |
| ⌘ + 点击 | 已锁定位置 | 解除该位置的锁定 |
| ⌘ + 点击 | 中央格 | 切换锁定功能的启用/停用状态 |
| ⌘ + ⌥ + 点击 | 任意位置 | 打开{{GLYPH_PICKER_TITLE_LOCK|}}选择器 |
| ⌘ + ⌥ + 点击 | 中央格 | 清除所有锁定 |
| ⌘ + ⇧ + 点击 | 中央格 | 保存当前锁定到{{PRESETS_WINDOW_TITLE|}} |

### 拖曳操作

| 操作 | 来源 | 效果 |
|-----|------|------|
| ⌘ + 拖曳 | 参考字位置 | 锁定到目标位置 |
| ⌘ + 拖曳 | 已锁定位置 | 移动锁定字 |
| ⌘ + ⌥ + 拖曳 | 任意位置 | 复制锁定（来源保留） |

## 中央格说明

中央格（位置 4）始终显示当前编辑的字符，**无法被锁定**。

> [!TIP]
> 使用 **⌘ + 拖曳**可将中央格字符复制到其他位置，建立该字符的锁定字。

## 锁定字面板

锁定字面板是独立的浮动窗口，专门用于显示和管理锁定字。

### 打开方式

- **工具栏**：点击锁定按钮（{{ICON_LOCK}}）
- **右键菜单**：在九宫格内容区按右键，选择「{{CONTEXT_MENU_SHOW_LOCKED_PANEL|}}」

### 面板操作

面板以 3×3 九宫格形式显示所有锁定字的配置：

| 操作 | 效果 |
|-----|------|
| 点击 | 选取该格子 |
| 双击 | 打开字符选择器修改该位置 |
| ⌘ + 点击 | 多选（切换选取状态） |
| ⇧ + 点击 | 矩形选取 |
| ⌘ + A | 全选所有已锁定的位置 |
| Delete | 删除所有选取的锁定字 |
| Enter | 打开字符选择器批量修改选取的位置 |

### 中央格切换

面板的中央格显示锁头图标，点击可切换锁定功能的启用/停用状态：

- **启用时**：显示锁定图标（{{ICON_LOCK}}）
- **停用时**：显示解锁图标（{{ICON_LOCK_OPEN}}）

## 显示控制

### 切换锁定字面板

工具栏的锁定按钮（{{ICON_LOCK}}）控制锁定字面板的显示：

- **已打开**：面板显示，提示文字为「{{LOCKED_PANEL_TOGGLE_HIDE|}}」
- **已关闭**：面板隐藏，提示文字为「{{LOCKED_PANEL_TOGGLE_SHOW|}}」

### 锁定数据持久化

锁定字数据保存在字型文件中（`GSFont.userData`），会随文件保存：

- 关闭并重新打开字型文件后，锁定设置会被恢复
- 不同字型文件有独立的锁定设置

## 使用技巧

### 建立比较基准

锁定一个标准字符（如「永」），观察其他字符与它的搭配效果。

### 检查设计一致性

锁定多个已完成的字符，确保新设计的字符与它们风格一致。

### 快速保存配置

设置好锁定字配置后，可通过以下方式保存到{{PRESETS_WINDOW_TITLE|}}：

- **⌘ + ⇧ + 点击**九宫格中央格
- 点击锁定字面板的**保存按钮**
