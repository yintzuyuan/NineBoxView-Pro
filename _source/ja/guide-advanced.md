## {{LIGHT_TABLE}} 統合

{{NINE_BOX_VIEW}} はサードパーティプラグイン **[{{LIGHT_TABLE}}](https://formkunft.com/light-table/)** との統合をサポートしており、9マス内で異なるバージョンの文字デザインを比較できます。

### {{LIGHT_TABLE}} とは？

{{LIGHT_TABLE}} は独立したGlyphsプラグインで、フォントバージョンの比較機能を提供します。{{LIGHT_TABLE}} がインストールされ、そのツールがアクティブな場合、9マス内で比較バージョンの文字を表示できます。

### 有効化条件

9マスで比較バージョンを表示するには、以下の条件を満たす必要があります：

1. **{{LIGHT_TABLE}} をインストール**：{{GLYPHS_PLUGIN_MANAGER}} から {{LIGHT_TABLE}} プラグインをインストール
2. **{{LIGHT_TABLE}} ツールを有効化**：GlyphsツールバーでL {{LIGHT_TABLE}} の比較ツールを選択
3. **比較バージョンを選択**：{{LIGHT_TABLE}} パネルで比較するバージョンを選択
4. **Shiftキーを押し続ける**：Shiftキーを押して比較モードをトリガー

### 使用方法

1. {{LIGHT_TABLE}} がインストールされ有効化されていることを確認
2. {{LIGHT_TABLE}} パネルで比較バージョンを選択
3. **Shift** キーを押し続ける
4. 9マス内の文字が比較バージョンに切り替わる

![Light Table 比較効果](../assets/images/feature-lighttable.gif)

> [!NOTE]
> **Shiftキーの動作**：{{LIGHT_TABLE}} がインストールされていないか、ツールが有効化されていない場合、Shiftを押し続けると **Soloモード** になり、参照文字とロック文字が一時的に非表示になり、現在編集中の文字のみが表示されます。周囲のセルの重ね合わせ効果と現在の文字自体の違いを素早く比較するのに便利です。

## キーボードショートカット

### 9マス内操作

| ショートカット | 操作 | 説明 |
|-------|------|------|
| クリック | ランダム配置 | 参照文字をランダムに再配置 |
| Cmd + クリック | ロック/ロック解除 | その位置のロック状態を切り替え |
| Cmd + 中央セルをクリック | ロック機能切り替え | ロック機能を有効/無効にする |
| Cmd + Option + クリック | グリフピッカー | グリフピッカーを開いてロック文字を設定 |
| Cmd + Option + 中央セルをクリック | 全ロックをクリア | すべての位置のロックを一度にクリア |
| Cmd + Shift + 中央セルをクリック | ロックプリセットを保存 | 現在のロック設定を{{PRESETS_WINDOW_TITLE|}}に保存 |

### ドラッグ操作

| ショートカット | ドラッグ効果 |
|-------|---------|
| なし | 位置を交換（9マス内）/ 挿入（編集ビュー） |
| Cmd | ロック文字操作（移動またはロック） |
| Option | 9マス内：参照文字をコピー；編集ビューへ：新しいタブで文字を開く |
| Shift | 新しいタブで9マスを開く |
| Cmd + Option | ロック文字をコピー（元は保持） |

### ツールバー操作

ツールバーボタンはクリックでトリガーされ、一部のボタンはマウスホバー時にツールチップを表示します：

| ボタン | 機能 |
|-----|------|
| {{ICON_SUN}} / {{ICON_MOON}} | ダーク/ライトモードを切り替え |
| {{ICON_PRESETS}} | {{PRESETS_WINDOW_TITLE|}}パネルを表示/非表示 |
| {{ICON_LOCK}} | {{LOCKED_PANEL_TITLE|}}パネルを表示/非表示 |
| {{ICON_REFERENCE}} | {{REFERENCE_PANEL_TITLE|}}パネルを表示/非表示 |

## グリッド線表示

グリッド線機能は9マスにガイドラインを表示し、文字の整列と比率の確認に役立ちます。

### 有効化

1. 9マス内で右クリック
2. 「{{CONTEXT_MENU_SHOW_GRID_LINES|}}」を選択

### グリッド内容

グリッド線は以下のガイドラインを表示します：

- **{{GRID_EM_BOX}}**：文字の境界ボックス
- **{{GRID_CENTER_LINE}}**：水平および垂直の中心線

### 無効化

1. 9マス内で右クリック
2. 「{{CONTEXT_MENU_HIDE_GRID_LINES|}}」を選択

> [!NOTE]
> グリッド線の設定は自動的に保存され、再起動後も維持されます。

## ドラッグ高度な操作

### Shift + ドラッグで新しいタブを開く

**Shift** キーを押しながら9マスからGlyphsウィンドウにドラッグすると、9マス全体の内容を新しいタブで開くことができ、編集ビューで配置効果を素早くプレビューするのに便利です。

## 特殊レイヤーサポート

Glyphsで非マスターレイヤーを編集している場合、9マスの周囲のセルは対応するレイヤーバージョンを自動的に同期表示し、特殊レイヤーと周囲の文字の組み合わせ効果をリアルタイムでプレビューできます。

> [!TIP]
> 周囲のセルを特殊レイヤーに追従させずにマスターレイヤーに保持したい場合は、「ロック文字」または「参照文字」機能を使用してください。ロック文字と参照文字はマスターレイヤーに維持されます。

## データ永続化

{{NINE_BOX_VIEW}} のデータは2つの場所に保存されます：

| 保存場所 | 内容 | 説明 |
|---------|------|------|
| フォントファイル | 参照文字、ロック文字 | ファイルと共に保存、ファイル間で共有されない |
| アプリケーション設定 | ウィンドウ設定、{{PRESETS_WINDOW_TITLE|}}、テーマなど | グローバル設定、ファイル間で使用可能 |

> [!TIP]
> インポート/エクスポート機能を使用して、異なるコンピュータ間で{{PRESETS_WINDOW_TITLE|}}を共有できます。
