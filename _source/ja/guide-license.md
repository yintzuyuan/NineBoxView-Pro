# ライセンスと試用

{{NINE_BOX_VIEW}} は14日間の無料試用期間を提供しており、すべての機能を十分に体験してからライセンス購入を決定できます。

## 試用期間

{{NINE_BOX_VIEW}} を初めて開くとウェルカムウィンドウが表示され、「{{LICENSE_WELCOME_START_TRIAL|}}」を選択して14日間の無料試用を開始します。

| 段階 | 説明 |
|-----|------|
| 試用中 | すべての機能を制限なく使用可能、残り日数は情報メニューに表示 |
| 試用終了 | ライセンスを購入、ライセンスキーを入力、または無料版に切り替え可能 |

## ライセンス購入

1. 情報メニューの「{{INFO_MENU_PURCHASE|}}」をクリック
2. 支払い完了後、ライセンスファイル（`.license`）がメールに送信されます

**ライセンスプラン**：一度の購入で永久使用、無料アップデート付き、お持ちの複数のコンピュータで使用可能。

## ライセンスの有効化

ライセンス購入後、`.license` ライセンスファイルを受け取ります。有効化方法：

1. タイトルバーの「{{ICON_MENU}}」ボタンをクリックし、「{{INFO_MENU_ENTER_LICENSE|}}」を選択
2. ライセンスファイルをウィンドウにドラッグ＆ドロップ（ヒント：「{{LICENSE_DROP_HINT|}}」）
3. 「{{LICENSE_DROP_ACTIVE|}}」が表示されたらマウスを放す
4. 検証成功後、「{{LICENSE_ACTIVATION_SUCCESS_TITLE|}}」が表示され、自動的に有効化が完了

### よくあるエラー

| エラーメッセージ | 説明 |
|---------|------|
| {{LICENSE_ERROR_INVALID_KEY|}} | ライセンスキーの形式が正しくないか、一致しない |
| {{LICENSE_ERROR_INVALID_FILE|}} | ライセンスファイルが破損または無効 |
| {{LICENSE_ERROR_WRONG_PRODUCT|}} | ライセンスキーがこの製品に適用されない |

## ライセンス状態

「{{ICON_MENU}}」→「{{ABOUT_MENU_ITEM|}}」をクリックして、ライセンス状態とバージョン情報を確認します。

## 無料版

{{NINE_BOX_VIEW}} を購入しない場合は、無料版の **NineBoxView**（Proサフィックスなし）を使用できます。

### 無料版とPro版の違い

| 機能 | 無料版 | Pro版 |
|-----|--------|--------|
| 基本9マスプレビュー | {{ICON_CHECK}} | {{ICON_CHECK}} |
| 参照文字機能 | {{ICON_CHECK}} | {{ICON_CHECK}} |
| ロック文字機能 | {{ICON_CHECK}} | {{ICON_CHECK}} |
| ドラッグ＆ドロップ機能 | {{ICON_X}} | {{ICON_CHECK}} |
| {{PRESETS_WINDOW_TITLE|}}機能 | {{ICON_X}} | {{ICON_CHECK}} |
| インポート/エクスポート | {{ICON_X}} | {{ICON_CHECK}} |
| {{LIGHT_TABLE}} 統合 | {{ICON_CHECK}} | {{ICON_CHECK}} |

### バージョンの切り替え

{{GLYPHS_PLUGIN_MANAGER}} から対応するバージョンをインストールし、旧バージョンを削除します。

## よくある質問

### 複数のコンピュータで使用できますか？

はい、お持ちの複数のコンピュータで同じライセンスキーを使用できます。新しいコンピュータでライセンスファイルを再度読み込む必要があります。ライセンスファイルは期限切れにならず、無制限に使用できます。

### ライセンスキーを紛失した場合は？

> [!TIP]
> [support@erikyin.net](mailto:support@erikyin.net) にメールし、購入時に使用したメールアドレスを提供してください。ライセンスキーの回復をお手伝いします。

### ライセンスデータはどこに保存されますか？

ライセンスデータはコンピュータに保存されます。オンライン時はライセンスの有効性のみを検証し、その他の使用情報は収集しません。Glyphsを再インストールした後、ライセンスファイルを再度読み込む必要があります。
