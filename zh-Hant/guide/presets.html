字組

字組功能讓您儲存常用的參考字組合或鎖定字配置，快速切換不同測試情境。

開啟字組面板

有兩種方式開啟字組面板：

  1. 工具列按鈕：點擊工具列的 按鈕
  2. 右鍵選單：在九宮格內按右鍵，選擇「顯示字組面板」

分頁說明

字組面板包含兩個分頁：

參考字分頁

儲存參考字的字符組合。每個字組包含：

  • 字組名稱
  • 參考字內容（字符清單）
  • 釘選狀態

鎖定字

儲存鎖定字的位置配置。每個字組包含：

  • 字組名稱
  • 各位置的鎖定字符
  • 釘選狀態

搜尋與排序

搜尋功能

在分頁控制項下方有搜尋欄位，可快速篩選字組：

  1. 在搜尋框輸入關鍵字
  2. 字組列表會即時過濾
  3. 搜尋不區分大小寫
  4. 清空搜尋框可恢復完整列表

提示

當搜尋無結果時，會顯示「找不到結果」提示。

排序功能

點擊面板底部的排序按鈕，可選擇排序方式：

排序選項 說明
名稱 A → Z 依名稱字母順序排列
名稱 Z → A 依名稱反向字母順序排列
加入日期（新→舊） 最新加入的字組在前
加入日期（舊→新） 最早加入的字組在前

備註

排序狀態會自動儲存，重啟後保持。

建立字組

建立參考字字組

方法一：從參考字面板儲存

  1. 在參考字面板輸入字符
  2. 點擊「儲存」按鈕
  3. 字組會以參考字內容作為顯示名稱

方法二：從字組面板新增

  1. 切換到「參考字」分頁
  2. 點擊面板底部的 按鈕
  3. 當前的參考字會儲存為新字組

建立鎖定字字組

方法一：快捷操作

  1. 設定好鎖定字配置
  2. 按住 Cmd + Option
  3. 點擊中央格
  4. 當前鎖定配置會儲存為新字組

方法二：從字組面板新增

  1. 切換到「鎖定字」分頁
  2. 點擊面板底部的 按鈕
  3. 當前的鎖定字配置會儲存為新字組

管理字組

載入字組

在字組面板中雙擊任一字組，即可載入該字組的內容。

重新命名

方法一：鍵盤快捷鍵

  1. 選擇字組
  2. 按下 Enter 鍵進入編輯模式
  3. 輸入新名稱

方法二：右鍵選單

  1. 在字組上按右鍵
  2. 選擇「重新命名…」
  3. 輸入新名稱

覆寫字組

  1. 修改當前的參考字或鎖定字配置
  2. 在要覆寫的字組上按右鍵
  3. 選擇「以當前取代」
  4. 確認覆寫

注意

覆寫操作無法復原，請確認後再執行。

刪除字組

方法一：鍵盤快捷鍵

  1. 選取字組（單選或多選）
  2. 按下 Delete 鍵

方法二：右鍵選單

  • 單一刪除：在字組上按右鍵，選擇「刪除」
  • 批次刪除：選取多個字組（Cmd + 點擊或 Shift + 點擊），按右鍵選擇「刪除 %lu 個項目」

右鍵選單

項目 說明
釘選 / 取消釘選 釘選或取消釘選
重新命名… 重新命名字組
以當前取代 以當前內容取代字組
刪除 刪除字組

釘選功能

釘選的字組會固定顯示在清單頂部，方便快速存取。

釘選操作

單一釘選：

  1. 在字組上按右鍵
  2. 選擇「釘選」

批次釘選：

  1. 選取多個字組
  2. 按右鍵，選擇「釘選 %lu 個項目」

取消釘選

  1. 在已釘選的字組上按右鍵
  2. 選擇「取消釘選」

排序規則

字組清單的排序遵循以下規則：

  1. 已釘選字組：永遠顯示在最上方
  2. 未釘選字組：依照排序設定排列

使用排序按鈕可切換四種排序方式。在各分組內（已釘選/未釘選），字組依照選定的排序方式排列。

匯入匯出

匯出字組

匯出選取的字組：

  1. 選取要匯出的字組
  2. 點擊面板底部的 按鈕
  3. 選擇「匯出選取字組⋯」
  4. 選擇儲存位置

匯出所有字組：

  1. 點擊面板底部的 按鈕
  2. 選擇「匯出所有字組⋯」
  3. 選擇儲存位置

匯出格式

匯出的檔案為 JSON 格式，依字組類型使用不同副檔名：

  • 參考字：.nineboxref
  • 鎖定字：.nineboxlock

檔案內容包含：

  • 格式標記
  • ID 編號
  • 字組資料

備註

匯出時不會保留釘選狀態。

匯入字組

  1. 點擊面板底部的 按鈕
  2. 選擇「匯入⋯」
  3. 選擇要匯入的檔案
  4. 選擇匯入模式：
    • 追加：追加到現有字組
    • 全部取代：取代所有現有字組

匯入注意事項

匯入對話框會依據當前分頁自動過濾檔案類型：

  • 在參考字分頁時，只顯示 .nineboxref 檔案
  • 在鎖定字分頁時，只顯示 .nineboxlock 檔案

匯入錯誤訊息

錯誤 說明
檔案格式無效。 檔案格式無效
此檔案由較新版本建立，請更新外掛。 檔案版本過新，請更新外掛
此檔案包含%@字組，但您目前在%@分頁。 檔案類型與當前分頁不符
此檔案中沒有有效的字組。 檔案中沒有有效的字組

空狀態提示

當分頁中沒有字組時，會顯示提示訊息：

  • 參考字分頁：「尚無參考字組 + 新增」
  • 鎖定字分頁：「尚無鎖定字組 + 新增」

使用技巧

為不同語言建立字組

建立多組測試字組，例如：

  • 「繁中常用」：永東國酬鷹
  • 「日文假名」：あいうえお
  • 「韓文」：가나다라마

為不同專案建立配置

使用匯入匯出功能在不同電腦或專案間共享字組：

  1. 在一台電腦上建立常用字組
  2. 匯出為檔案
  3. 在其他電腦匯入

搭配鎖定字建立完整測試環境

  1. 建立參考字字組：常用參考字
  2. 建立鎖定字字組：標準字位置配置
  3. 快速載入兩者，建立完整測試環境
