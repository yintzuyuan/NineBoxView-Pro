## üöß About This Repository

**NineBoxView Pro** is a complete rewrite of the original python plugin, now built with **Objective-C** for native performance.

This repository currently hosts the **documentation** and **issue tracker** for the Pro version. The binary release will be available here shortly.

### Key Features
* **Native Drag & Drop:** Intuitive interaction with zero lag.
* **Smart Locking:** Pin reference glyphs for precise spacing adjustments.
* **Light Table Sync:** Seamless integration with the official Light Table plugin.
* **CJK Optimized:** Designed specifically for Hanzi, Kanji, and Hangul workflow.
